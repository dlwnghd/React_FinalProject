## 😺 소개
네고마켓은 중고 거래 커머스 플랫폼입니다. <br>채팅으로 중고 및 무료 상품 거래가 가능하며, 시세 동향 페이지에서 상품의 평균 시세를 일별로 확인할 수 있습니다. 또한, 개인 가계부를 통해 손쉽게 자신의 거래 내역을 확인할 수 있어 신뢰성을 높이고 편리한 거래 서비스을 제공합니다.
